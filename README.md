# clinvar-variant-analysis

Analysis of pathogenic variants from ClinVar using Python
This project analyzes pathogenic genetic variants from the ClinVar database using Python.

## Data

ClinVar variant summary dataset:
https://ftp.ncbi.nlm.nih.gov/pub/clinvar/

## Analysis

- Load ClinVar dataset
- Filter pathogenic variants
- Count variants per gene
- Visualize most affected genes
